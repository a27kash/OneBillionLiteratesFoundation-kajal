About

Fun2Learn is an offline desktop application created using open softwares to run effictively on Ubuntu Operating Environments, though it can just be ported to any other environment. All you need to get this application started is a local running server on your environment and obviously the downloaded bundle. The download includes the complete source files including HTML, CSS, and JavaScript as well as optional LESS stylesheets for customization.

Whether you're a student looking to practice your learning, a teacher looking for attractive and efficient classroom topics, or a test maker looking to create not same but similar tests on the exhaustive topics, this application is to your rescue!
